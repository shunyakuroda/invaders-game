# インベーダーゲーム

これは、HTML、CSS、およびJavaScriptで作成されたシンプルなインベーダースタイルのゲームです。

## セットアップ

特別なセットアップは必要ありません。ウェブブラウザで `index.html` ファイルを開くだけでゲームを開始できます。

## 遊び方

- **移動**: プレイヤーの宇宙船を動かすには、左矢印キーと右矢印キーを使用します。
- **射撃**: 弾を撃つには、スペースキーを押します。
- **目的**: 敵が画面の下に到達する前に、すべての敵を破壊してください。

## ゲームオーバー

敵があなたの宇宙船に到達すると、ゲームオーバーです。

## 勝利

すべての敵を破壊すると、ゲームに勝利します。
